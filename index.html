最新のCSS記法まとめ

モダンCSS技術とベストプラクティス

概要

このページでは、最新のCSSの機能や記法について紹介します。コードと実例を用いて、モダンなCSSテクニックを解説します。

CSS変数（カスタムプロパティ）

カスタムプロパティ（CSS変数）を使用すると、再利用可能な値をドキュメント全体で定義して使用できます。

:root {
  --primary-color: #4a6da7;
  --spacing-unit: 1rem;
}

.element {
  color: var(--primary-color);
  padding: var(--spacing-unit);
}

このページでは、CSS変数を使って色やスペーシングなどを一元管理しています。変数は--変数名で定義し、var(--変数名)で使用します。

CSS Grid Layout

CSSグリッドは、行と列の両方向にレイアウトを制御できる強力な2次元レイアウトシステムです。

.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  grid-gap: 1rem;
}

CSS Flexbox

フレックスボックスは、コンテナ内のアイテムの配置と分配を柔軟に制御できる1次元レイアウトモデルです。

.flex-container {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}

.flex-item {
  flex: 1 1 200px;
}

アニメーションとトランジション

CSSアニメーションとトランジションを使用すると、JavaScriptを使わずに要素をアニメーション化できます。

@keyframes pulse {
  0% { transform: scale(1); }
  50% { transform: scale(1.05); }
  100% { transform: scale(1); }
}

.animated-box {
  animation: pulse 2s infinite;
  transition: background-color 0.3s;
}

このボックスはアニメーション中です！

ホバーすると色が変わります

メディアクエリ

メディアクエリを使用すると、デバイスの特性（画面サイズなど）に基づいてスタイルを適用できます。

@media (max-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr;
  }
}

ブラウザのウィンドウサイズを変更すると、このページのレイアウトが調整されます。

論理プロパティ

論理プロパティは、書字方向に依存しないレイアウトを作成するのに役立ちます。

.element {
  margin-block: 1rem;
  padding-inline: 1rem;
  border-inline-start: 1px solid black;
}

この要素は論理プロパティを使用しています。

従来のmargin-topの代わりにmargin-block-start、padding-leftの代わりにpadding-inline-startのように使用します。

コンテナクエリ

コンテナクエリを使用すると、親要素のサイズに基づいてスタイルを適用できます。

.container {
  container-type: inline-size;
}

@container (min-width: 400px) {
  .child {
    background-color: blue;
  }
}

このコンテナの角をドラッグしてリサイズしてください ↘

コンテナの幅が400px以上になると色が変わります

CSS ネスティング

CSSネスティングでは、セレクタを入れ子にすることができ、コードの可読性と保守性が向上します。

.parent {
  color: blue;
  
  & .child {
    color: red;
    
    &:hover {
      color: green;
    }
  }
}

モダンセレクタ

CSSには、要素の選択方法を柔軟にするための新しいセレクタがあります。

/* :is() セレクタ */
:is(h1, h2, h3):hover { color: purple; }

/* :where() セレクタ (低い特異性) */
:where(section, article) p { line-height: 1.6; }

/* :has() 関係セレクタ */
div:has(> img) { padding: 1rem; }
